# ZLHMA-WAE Crypto Trend Breakout

This example demonstrates a breakout strategy using the **Zero‑Lag Hull Moving Average** and **Waddah Attar Explosion** indicators. The strategy is configured for a 1‑hour chart with confirmation from a higher timeframe (default 4‑hour).

Key features:

- Trend bias from the higher‑timeframe ZLHMA.
- Momentum breakout from WAE.
- ATR and Choppiness filters to control volatility and noise.
- Optional funding rate filter.
- Risk based position sizing with trailing stop and take‑profit.

The Pine Script is intended as an educational example. Refer to the [TradingView documentation](https://www.tradingview.com/pine-script-docs/) for full details on available functions and strategy behavior.
